## 常用命令介绍

::: tip 提示
我使用的`pnpm`包管理工具，你还可以使用`yarn`或者`npm`
:::

+ `pnpm install`

初始化项目安装依赖

+ `pnpm dev`

启动项目

+ `pnpm build:test`

测试环境打包

+ `pnpm build`

生产环境打包

+ `pnpm lint`

语法校验

+ `pnpm new`

生产模板

> 这里只是做简单介绍，具体详细介绍请看[配置参考scripts](/config/scripts.md)
